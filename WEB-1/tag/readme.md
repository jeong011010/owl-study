## Tag

HTML 언어를 설명하는 도구.  
```html
<태그> 내용 </닫는태그>
```
위와 같이 사용

## tag 종류

- h1-h6 : 제목 (heading)
- strong : 강조
- u : 밑줄 (underline)

- p : 단락 (paragraph)
- br : 줄바꿈

## tag 특징

<p>와 <br>은 줄바꿈의 역할을 하지만 다르다.  
<p>태그는 여는 태그와 닫는 태그 두개를 사용하지만 <br>태그는 하나만 사용하면 줄바꿈이 된다.  
그 부분에서는 <br> 태그가 편리하지만 <p>와 같이 여닫는 태그에는 style을 적용시킬 수 있다.
